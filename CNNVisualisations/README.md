This project was undertaken to investigate and demonstrate a selection of common CNN visualisation methods. 

Please see [CNNVisualisationsReport.pdf](https://github.com/fin-brown/misc-msc-projects/blob/master/CNNVisualisations/CNNVisualisationsReport.pdf) for full details.

The methods included are:
- Class Activated Heatmaps
- Guided Backpropagation
- Occlusion Sensitivity
